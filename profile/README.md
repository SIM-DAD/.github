## 📦 Projects

### [TASS](https://github.com/SIM-DAD/tass) — *In Development*
**Text Analysis · Desktop App · Windows**

A native Windows desktop application designed to make professional-grade text analysis accessible without Python, R, or a command line. Targets dictionary-based scoring, ML classification, group statistical comparisons, and publication-quality visualizations in a minimal GUI — built for domain experts who need results, not toolchain overhead.

---

### [The Cage](https://github.com/SIM-DAD/the-cage) — *Launching Soon*
**Next.js · PostgreSQL · MySQL · Docker · SSO**

A web application for equipment reservation and QR-based checkout. Features dual-database architecture (PostgreSQL + MySQL via Prisma ORM), Docker Compose orchestration, REST API, Microsoft Entra SSO, and Snipe-IT asset management integration.

---

### [Open WebUI Launcher](https://github.com/SIM-DAD/open-webui-launcher) — *Available Now*
**Docker · Ollama · Local LLMs · Desktop**

A desktop launcher for running Open WebUI locally via Docker + Ollama. Handles hardware detection, model selection, and safe configuration defaults — designed to make local LLM deployment accessible to non-technical users, with classroom-safe defaults for instructor-guided use.
